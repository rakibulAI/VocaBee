VocaBee 🐝

A Demo English Learning Platform for Bangladeshi Students

VocaBee is a demo website designed to simulate an online platform where students can learn English vocabulary and purchase structured English learning courses.
The project focuses on simplicity, beginner-friendly design, and easy pronunciation for Bangladeshi learners.

This project is created for portfolio and learning purposes to demonstrate frontend development skills.

---

✨ Features

- 📚 English vocabulary learning concept
- 🎓 Course showcase page
- 💳 Course purchase interface (demo)
- 📱 Mobile responsive layout
- ⚡ Fast and lightweight design
- 🇧🇩 Designed for Bangladeshi learners

---

🛠️ Built With

- HTML5
- CSS3
- JavaScript
